1. Project Goal
Layout:

Left Panel: Image with slogan (hidden on mobile).

Right Panel: Signup form with inputs, social logins, and T&C checkbox.

Responsive:

Mobile: Only form.

Tablet/Desktop: Side-by-side image + form.

UI Match: Follow reference design for spacing, colors, typography.

2. Layout & Typography
tsx
-
-
<div className="text-center mb-12">
  <h1 className="font-bold text-3xl sm:text-4xl md:text-5xl text-white mb-4">
    Create an account
  </h1>
  <p className="text-gray-400 mb-8">
    Already have an account? <a href="#" className="text-white hover:underline">Log in</a>
  </p>
</div>
Key Points:

Use responsive font sizes (sm:, md: breakpoints).

Add consistent spacing between title, subtitle, and form.

Wrap in a centered container with margin/padding.
